# Mobile UI Design System

## Design Principles
- Clean, modern, minimal
- Mobile-first (iOS + Android)
- Accessible (WCAG AA)

## Colors
- Primary: #4F46E5
- Secondary: #22C55E
- Background: #0F172A
- Surface: #1E293B
- Text Primary: #FFFFFF
- Text Secondary: #94A3B8
- Error: #EF4444

## Typography
- Heading: Bold, large
- Body: Regular
- Small: Muted

## Spacing Scale
- xs: 4
- sm: 8
- md: 16
- lg: 24
- xl: 32

## Components Rules
- Buttons: rounded-lg, consistent padding
- Inputs: clear borders, focus states
- Cards: soft shadows, padding
- Use consistent spacing everywhere

## Dark Mode
- Default ON
