# UI Component Rules

## General
- All components must be reusable
- Accept props for customization
- Use TypeScript interfaces

## Button
- Variants: primary, secondary, outline
- States: loading, disabled

## Input
- Label + error message support
- Controlled inputs

## Cards
- Padding: 16
- Rounded corners
- Optional shadow

## Lists
- FlatList optimized
- Key extractors required
